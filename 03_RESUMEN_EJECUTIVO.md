## 🎯 QUÉ ES IA LAB

Sistema completo y profesional para:

✅ **Relevar** madurez en IA de todo el equipo GrupoCESA
✅ **Calcular** score automático (0-100) y nivel (Principiante/Intermedio/Avanzado)
✅ **Identificar** barreras y necesidades de capacitación
✅ **Personalizar** talleres por grupo/área
✅ **Recopilar** datos en Google Sheet para análisis
✅ **Enviar** confirmación automática por email

## 🎯 DECISIONES TÉCNICAS

### Por qué elegimos esto

```
Google Apps Script (no Node.js/Python):
→ Integración nativa con Google Sheets
→ No requiere servidor
→ Mantenimiento mínimo
→ Costo: $0

Google Sheets (no base de datos):
→ Suficiente para volúmenes normales
→ Análisis integrado (Pivot, Gráficos)
→ Acceso compartible
→ Backup automático
→ Costo: $0

HTML Vanilla (no React/Vue):
→ Sin dependencias
→ Funciona offline
→ Rendimiento óptimo
→ Menos código
→ Más seguro

Gmail (no SendGrid):
→ Ya incluido con Google
→ Confiable
→ No requiere API externa
→ Costo: $0
```
